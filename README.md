# 🎌 Anime Next Watch Generator

A simple web app that helps users decide what anime to watch next by showcasing trending anime, allowing genre-based filtering, and providing quick synopsis previews.

---

### 🎯 Goal

Build the foundation of the app with:

* Basic project setup
* API integration with Jikan
* Display of trending anime list

---

## 🧱 Tech Stack

* **Frontend:** HTML,  JS
* **Styling:** CSS
* **API:** [Jikan API](https://docs.api.jikan.moe)
* **Deployment:** Vercel (planned)

---

## 📦 Features To be Implemented

✅ Fetch trending anime from Jikan API
✅ Display anime in a clean card layout
✅ Show basic details:

* Title
* Poster Image
* Rating

---

## 🔗 API Used

### Get Top Anime

```
https://api.jikan.moe/v4/top/anime
```

##

## 🧠 How It Works

1. App loads and sends a request to Jikan API
2. Fetches top anime list
3. Stores data in state
4. Maps through data to render anime cards

---
